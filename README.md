# WeatherWise

Developed a weather forecasting application that provides real-time weather updates and forecasts based on user-selected locations. Integrated a weather API to fetch live data and display temperature, humidity, wind speed, and weather conditions through a responsive interface.

# Key Features:

Real-time weather updates  
Location-based weather search  
Temperature, humidity, and wind speed display  
Weather condition forecasts  
Responsive and user-friendly design  
API integration for live weather data  

# Technologies Used:

HTML  
CSS  
JavaScript  
React.js  
Weather API (OpenWeatherMap)  
Git & GitHub
